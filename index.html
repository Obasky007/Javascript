History of JavaScript

JavaScript was first created in 1995 by Brendan Eich at Netscape. It was originally called Mocha, then renamed to LiveScript, before becoming JavaScript. It started as a simple scripting language and has grown into a powerful tool for web development.

ES1 (1997)

The first official JavaScript standard by ECMA.

  • Basic types: Number, String, Boolean, Object
  • Functions and prototypes
  • Control flow: if/else, for, while
var x = 5; var y = 10; function add(a, b) { return a + b; } add(x, y); // 15

ES2 (1998)

Mainly minor editorial updates to align with ISO/IEC standards.

ES3 (1999)

The version that made JavaScript widely adopted.

  • Regular Expressions
  • try/catch for error handling
  • do...while loops
  • Improved String methods
var pattern = /js/i; console.log("JavaScript".match(pattern)); try { JSON.parse("bad json"); } catch (e) { console.log("Error caught"); }

ES4 (Abandoned in 2008)

Planned to add classes, modules, and more, but was too ambitious and got dropped.

ES5 (2009)

Brought modern features and strict mode.

  • Strict mode ("use strict")
  • JSON.parse / JSON.stringify
  • Array methods: map, forEach, filter, reduce
  • Object.defineProperty()
'use strict'; var nums = [1, 2, 3]; var doubled = nums.map(function(n){ return n * 2; }); console.log(doubled); // [2, 4, 6]

ES6 (2015)

The “big upgrade” that shaped modern JavaScript.

  • let / const for block scope
  • Arrow functions
  • Template literals
  • Classes and inheritance
  • Modules (import/export)
  • Promises for async
  • Destructuring and Spread operator
const greet = (name) => `Hello, ${name}!`; console.log(greet("Oba")); class Person { constructor(name) { this.name = name; } sayHi() { return `Hi, I'm ${this.name}`; } } const me = new Person("Oba"); console.log(me.sayHi());
