History of JavaScript

Created in 1995 by Brendan Eich at Netscape, JavaScript evolved from a small scripting tool into a powerful foundation of the modern web.

ES1 (1997)

The first official JavaScript standard by ECMA.

  • Basic types: Number, String, Boolean, Object
  • Functions and prototypes
  • Control flow: if/else, for, while
var x = 5; var y = 10; function add(a, b) { return a + b; } add(x, y); // 15

ES2 (1998)

Minor updates for ISO/IEC compliance.

ES3 (1999)

This version made JavaScript truly mainstream.

  • Regular Expressions
  • try/catch for error handling
  • do...while loops
  • Improved String methods
var pattern = /js/i;
console.log("JavaScript".match(pattern));
try { JSON.parse("bad json"); } catch (e) { console.log("Error caught"); }

ES4 (Abandoned 2008)

Planned to add classes, modules, and more — but dropped for being too ambitious.

ES5 (2009)

Introduced modern core features and strict mode.

  • Strict mode ("use strict")
  • JSON.parse / JSON.stringify
  • Array methods: map, forEach, filter, reduce
  • Object.defineProperty()
'use strict';
var nums = [1, 2, 3];
var doubled = nums.map(function(n){ return n * 2; });
console.log(doubled); // [2, 4, 6]

ES6 (2015)

The major update that transformed JavaScript into what we know today.

  • let / const
  • Arrow functions
  • Template literals
  • Classes and inheritance
  • Modules (import/export)
  • Promises for async
  • Destructuring and Spread operator
const greet = (name) => `Hello, ${name}!`;
console.log(greet("Oba"));

class Person {
  constructor(name) { this.name = name; }
  sayHi() { return `Hi, I'm ${this.name}`; }
}
const me = new Person("Oba");
console.log(me.sayHi());
